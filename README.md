# Nextera MED Static Portfolio

Standalone static page prepared for GitHub Pages.

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `script.js`: portfolio cards and video embed logic
- `assets/nextera-med-facebook-profile.jpg`: public profile image fetched from Facebook

## Add Portfolio Videos

Open `script.js` and replace the empty `url` values inside `PORTFOLIO_ITEMS` with:

- YouTube links
- Vimeo links

The page will automatically convert supported links into embeds.

## Publish To GitHub Pages

Push this folder as its own repository, then enable GitHub Pages from the repository settings using the main branch root.
